Trabajo Práctico: Conexión Remota y Administración

Harkaitz Herrero Carnicero

EJERCICIO 1: Creación usuario Ubuntu y conexiones SSH

En este ejercicio creamos un usuario en Ubuntu, instalamos el servidor ssh para conectarnos con otros equipos Linux, nos conectamos con Putty desde Windows a un equipo Linux, y finalmente lo configuramos para acceder sin tener que poner la contraseña.

  • Crear usuario
  • conexión servidor SSH
  • Conexión con Putty
  • Conexión sin contraseña con keygen

EJERCICIO 2: Conexión escritorio remoto con Remmina y Windows

Levantamos el servicio de escritorio remoto en Windows, y establecemos una conexión con otro Windows de la misma red, y otra desde Linux usando el porgrama Remmina.

  • Escritorio Remoto en Windows
  • Conectar desde Linux usando Remmina (configuración de IP y conexión)

EJERCICIO 3: Administración remota con TeamViewer

Instalamos y hacemos conexiones con Teamviewer en Ubuntu, Windows y un telefono movil.

  • Conexión remota Windows-Windows
  • Conexión desde Windows a Ubuntu
  • Conexión Ubuntu-Ubuntu
  • Conexión desde movil a Windows
  • Conexión desde movil a Ubuntu
